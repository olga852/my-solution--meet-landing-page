### Что я изучила

Центрование изображения

```css
.img-responsive {
  height: 153px;
  position: relative;
  left: 50%;
  transform: translate(-50%, 0);
}
```

Как правильно использовать overflow-x с мобильными устройствами

```css
html,
body {
  overflow-x: hidden;
}
```
